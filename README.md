## Description

This repository is assignment (backend) for Data Wow company
You can run and test follow command below


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev


```

## When Running


When running you can see Api document in swagger by

[http://localhost:3001/api/](http://localhost:3001/api/)




## Test

```bash

# e2e tests
$ npm run test:e2e

```
